## Overview
This qualitative study analyzes social factors and mechanisms perpetuating criminal activities (e.g., theft, robbery, drug abuse, assault) in Kumasi's informal settlements (Ayigya, Oforikrom, Asawase), based on 46 interviews with suspects, police, religious leaders, ex-convicts, guardians, and assembly members.
